Quick Start Path
----------------
1. Read START_HERE.md
2. Run: ./build.sh
3. Check examples/basic-app/
4. Add library to your project
5. Import SecurityConfig and CorsConfig

File Purposes
------------
START_HERE.md           - Overview and quick start
GETTING_STARTED.md      - Detailed walkthrough with examples
QUICK_REFERENCE.md      - Quick commands and snippets
CONVERSION_SUMMARY.md   - Complete conversion documentation
INSTALLATION_GUIDE.md   - Advanced setup and troubleshooting
PROJECT_SUMMARY.md      - Technical details and architecture
build.sh                - One-command build script
pom.xml                 - Library Maven configuration
SecurityConfig.java     - JWT authentication and security rules
CorsConfig.java         - Cross-origin request configuration
examples/basic-app/     - Working reference implementation

Build & Install
---------------
./build.sh
# or
mvn clean install

Artifact Location
----------------
~/.m2/repository/com/example/keycloak-spring-security-lib/1.0.0/

Generated Files (after build)
-----------------------------
target/keycloak-spring-security-lib-1.0.0.jar          (main library)
target/keycloak-spring-security-lib-1.0.0-sources.jar  (source code)
target/keycloak-spring-security-lib-1.0.0-javadoc.jar  (documentation)
